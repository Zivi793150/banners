Техническое задание: Телеграм бот "Ответы на вопросы"

1. Введение

Цель проекта состоит в создании телеграм бота "Ответы на вопросы", который будет предоставлять подробные рекомендации на основе данных, подготовленных из роликов YouTube. Бот должен обрабатывать вопросы пользователей, искать информацию в базе данных и использовать GPT-4 сеть для формирования ответов и задания дополнительных вопросов.

2. Основные характеристики бота

2.1. Функциональные возможности

- Бот получает текстовые сообщения от пользователей.
- Обрабатывает входящие сообщения с помощью программы роутинга (перенаправление данных).
- Ищет информацию в предварительно подготовленной базе данных, которая должна быть векторизована.
- Взаимодействует с GPT-4 для формирования текстовых ответов и дополнительных вопросов.
- Выводит результаты в виде текстовых сообщений для пользователя.
- К тексту ответа добавляется скриншот и ссылка на видеоролик YouTube, использованные для формирования ответа.
- При возможности ссылка с временем из ролика и скриншот должны быть максимально близки к тексту ответа.
- Запоминает историю чата, включая заданные вопросы и ответы.
- Бот ведет диалог от имени "Родиона Чиркова — специалиста по созданию домашних кинотеатров и стерео систем для настоящих ценителей музыки и кино".
- Предлагает пользователю бесплатный лимит - 10 вопросов в день.
- При превышении лимита предлагает оплатить подписку на месяц или сделать разовый платеж за дополнительные вопросы.

2.2. Технологии

- Бот будет разработан для платформы Telegram, используя Telegram Bot API.
- Реализация программы роутинга (перенаправление данных) для обработки входящих запросов от пользователя.
- База данных должна быть предварительно подготовлена и векторизована для быстрого поиска информации.
- Для генерации ответов на вопросы пользователей будет использоваться нейронная сеть ChatGPT.
- Для монетизации и ограничения количества бесплатных вопросов будет использоваться система платежей YooKassa. (https://yookassa.ru/developers/)

3. Алгоритм работы бота

1. Пользователь запускает бота в Telegram и отправляет вопрос в текстовом формате.
2. Входящее сообщение передается на обработку программе роутинга (перенаправление данных).
3. Программа роутинга (перенаправление данных) проводит предварительную фильтрацию и обработку входящего вопроса, чтобы убрать ненужные символы и сделать текст единообразным.
4. После обработки вопроса бот осуществляет поиск информации в предварительно подготовленной базе данных с помощью векторных файлов.
5. Если вопрос найден в базе данных, бот формирует ответ и отправляет его пользователю в виде сообщения.
6. Если вопрос не найден, бот задает дополнительный вопрос, логически связанный с вопросом пользователя, чтобы собрать достаточно информации для формирования ответа.
7. Ответ пользователя на дополнительный вопрос с историей чата и необходимой информацией из базы данных отправляется в нейронную сеть.
8. Нейронная сеть генерирует ответ на основе полученных данных и отправляет его пользователю.

4. Монетизация

- Пользователям предоставляется возможность задать 10 вопросов бесплатно в течение дня.
- После превышения бесплатного лимита пользователю предлагается оплатить подписку на месяц или сделать разовый платеж за 30 дополнительных вопросов.
- Система платежей будет интегрирована с Telegram или другой платежной системой для удобства пользователей.

- Работы производятся на сервере заказчика, на котором должна быть установлена «чистая» виртуальная машина Ubuntu 20.04.
- По завершении проекта будет предоставлен готовый телеграм бот с описанным функционалом.
- Бот будет задокументирован, и будет предоставлена инструкция по его использованию.
- Весь код программы размещается на сервере заказчика.
- Проверка работоспособности бота будет проведена на тестовом сервере.
